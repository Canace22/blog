常用的Unix终端命令

一、文件与目录操作

  1. cd - 切换目录

  2. pwd - 显示当前工作目录的路径

  3. mkdir - 创建目录

  4. ls - 查看目录内容

  5. rm - 删除文件或目录

  6. cp - 复制文件或目录

  7. mv - 移动或重命名文件或目录

  8. touch - 创建空文件

二、文件内容查看与编辑

  1. cat - 查看文件内容

  2. less - 分页查看文件内容（适合大文件）

  3. head / tail - 查看文件的开头或结尾部分

  4. grep - 在文件中搜索指定的文本

  5. vim - 终端文本编辑器

三、调试与日志

  1. 检查端口占用情况

  2. kill - 终止进程

  3. ps - 查看当前运行的进程

  4. top/htop - 实时查看系统资源使用情况

  5. 查看日志

四、网络相关

  1. curl - 发送 HTTP 请求

  2. ping - 测试网络连接

  3. scp - 通过 SSH 复制文件

五、环境与配置

  1. export - 设置环境变量

  2. source - 加载配置文件

  3. alias - 创建命令别名

六、其他

  1. chmode - 修改文件权限

  2. 清空终端

  3. 查看命令历史

  4. 查找文件

  5. 压缩与解压

  6. ssh - 连接服务器

  7. du - 查看目录大小
