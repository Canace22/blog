一、文件模糊搜索

按下 t 键，会触发文件的模糊搜索功能，在输入框输入对应的字段，即可搜到文件,试了一下，gitlab 也同样适用。

二、目录相关扩展

安装 Octotree 扩展，可以像在开发工具中浏览文件一样浏览 github 上的项目，他跟 Sourcegraph 的不同在于更加快，而且不用打开新的页面，更加轻巧。
