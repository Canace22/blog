дёҖгҖҒеёёз”Ёж¶ҲжҒҜжҺЁйҖҒж–№жЎҲжҰӮиҝ°

Web з«Ҝе®һзҺ°жӣҙж–°еј№зӘ—зҡ„ж–№ејҸдё»иҰҒжңүд»ҘдёӢеҮ з§ҚпјҡиҪ®иҜўгҖҒй•ҝиҪ®иҜўгҖҒWebSocketгҖҒServer-Sent EventsпјҲSSEпјүд»ҘеҸҠ Service Worker + Push APIпјҲPWA жҺЁйҖҒпјүпјҢеҮ з§Қж–№жЎҲеҗ„жңүдјҳеҠЈпјҢдёӢйқўеҒҡдёҖдёӘз®ҖеҚ•еҜ№жҜ”гҖӮ

1. иҪ®иҜўпјҲPollingпјү

  • еҺҹзҗҶпјҡWebз«Ҝе®ҡж—¶еҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒиҜ·жұӮпјҢиҜўй—®жҳҜеҗҰжңүжӣҙж–°гҖӮ
  • е®һзҺ°пјҡдҪҝз”Ё setInterval жҲ– setTimeout е®ҡжңҹеҸ‘йҖҒ HTTP иҜ·жұӮпјҢжңҚеҠЎеҷЁиҝ”еӣһжӣҙж–°зҠ¶жҖҒжҲ–ж•°жҚ®гҖӮ
  • дјҳзӮ№пјҡе®һзҺ°з®ҖеҚ•пјҢе…је®№жҖ§еҘҪгҖӮ
  • зјәзӮ№пјҡе»¶иҝҹй«ҳпјҲдҫқиө–иҪ®иҜўй—ҙйҡ”пјүгҖҒжөӘиҙ№еёҰе®ҪгҖҒеўһеҠ жңҚеҠЎеҷЁеҺӢеҠӣгҖӮ
  • йҖӮз”ЁеңәжҷҜпјҡдҪҺйў‘жӣҙж–°жҲ–еҜ№е®һж—¶жҖ§иҰҒжұӮдёҚй«ҳзҡ„еңәжҷҜгҖӮ

2. й•ҝиҪ®иҜўпјҲLong Pollingпјү

  • еҺҹзҗҶпјҡWebз«ҜеҸ‘иө·иҜ·жұӮеҗҺпјҢжңҚеҠЎеҷЁдҝқжҢҒиҝһжҺҘзӣҙеҲ°жңүж•°жҚ®жҲ–и¶…ж—¶пјҢеҶҚйҮҚж–°еҸ‘иө·иҜ·жұӮгҖӮ
  • е®һзҺ°пјҡWebз«ҜеҸ‘йҖҒиҜ·жұӮпјҢжңҚеҠЎеҷЁжҢӮиө·зӣҙеҲ°жңүж•°жҚ®пјҢиҝ”еӣһж•°жҚ®еҗҺWebз«Ҝз«ӢеҚіеҸ‘иө·ж–°иҜ·жұӮгҖӮ
  • дјҳзӮ№пјҡжҜ”иҪ®иҜўе®һж—¶жҖ§жӣҙй«ҳпјҢеҮҸе°‘ж— ж•ҲиҜ·жұӮгҖӮ
  • зјәзӮ№пјҡжңҚеҠЎеҷЁйңҖеӨ„зҗҶеӨ§йҮҸжҢӮиө·иҝһжҺҘпјҢеҸҜиғҪеўһеҠ иө„жәҗж¶ҲиҖ—гҖӮ
  • йҖӮз”ЁеңәжҷҜпјҡдёӯзӯүе®һж—¶жҖ§йңҖжұӮгҖӮ

3. WebSocket

  • еҺҹзҗҶпјҡеҹәдәҺ TCP зҡ„е…ЁеҸҢе·ҘйҖҡдҝЎеҚҸи®®пјҢе»әз«ӢжҢҒд№…иҝһжҺҘеҗҺеҸҢж–№еҸҜдё»еҠЁжҺЁйҖҒж¶ҲжҒҜгҖӮ
  • е®һзҺ°пјҡWebз«ҜйҖҡиҝҮ new WebSocket(url) е»әз«ӢиҝһжҺҘпјҢжңҚеҠЎеҷЁзӣ‘еҗ¬ WebSocket иҜ·жұӮе№¶з»ҙжҠӨиҝһжҺҘжұ пјҢWebз«ҜйҖҡиҝҮ onmessage дәӢд»¶жҺҘж”¶ж•°жҚ®гҖӮ
  • дјҳзӮ№пјҡе®һж—¶жҖ§ејәгҖҒдҪҺе»¶иҝҹгҖҒиҠӮзңҒеёҰе®ҪгҖӮ
  • зјәзӮ№пјҡйңҖжңҚеҠЎеҷЁж”ҜжҢҒ WebSocket еҚҸи®®пјҲеҰӮ Node.js зҡ„ ws еә“пјүпјҢеӨҚжқӮеәҰиҫғй«ҳгҖӮ
  • йҖӮз”ЁеңәжҷҜпјҡй«ҳйў‘еҸҢеҗ‘йҖҡдҝЎгҖӮ

4. Server-Sent EventsпјҲSSEпјү

  • еҺҹзҗҶпјҡеҹәдәҺ HTTP зҡ„еҚ•еҗ‘йҖҡдҝЎпјҢжңҚеҠЎеҷЁеҸҜдё»еҠЁжҺЁйҖҒж•°жҚ®еҲ°Webз«ҜгҖӮ
  • е®һзҺ°пјҡWebз«ҜйҖҡиҝҮ EventSource API и®ўйҳ…жңҚеҠЎеҷЁдәӢд»¶жөҒпјҢжңҚеҠЎеҷЁиҝ”еӣһ Content-Type: text/event-stream зҡ„е“Қеә”пјҢжҢҒз»ӯеҸ‘йҖҒж•°жҚ®гҖӮ
  • дјҳзӮ№пјҡз®ҖеҚ•пјҲд»…йңҖ HTTPпјүгҖҒиҮӘеҠЁйҮҚиҝһгҖҒиҪ»йҮҸзә§гҖӮ
  • зјәзӮ№пјҡд»…ж”ҜжҢҒж–Үжң¬ж•°жҚ®гҖҒеҚ•еҗ‘йҖҡдҝЎпјҲжңҚеҠЎеҷЁеҲ°Webз«ҜпјүгҖӮ
  • йҖӮз”ЁеңәжҷҜпјҡеҚ•еҗ‘е®һж—¶жӣҙж–°гҖӮ

5. Service Worker + Push APIпјҲPWA жҺЁйҖҒпјү

  • еҺҹзҗҶпјҡйҖҡиҝҮжөҸи§ҲеҷЁеҗҺеҸ°жңҚеҠЎпјҲService WorkerпјүжҺҘж”¶жңҚеҠЎеҷЁжҺЁйҖҒпјҢеҚідҪҝйЎөйқўе…ій—ӯд№ҹиғҪжҳҫзӨәйҖҡзҹҘгҖӮ
  • е®һзҺ°пјҡWebз«ҜжіЁеҶҢ Service Worker е№¶иҜ·жұӮйҖҡзҹҘжқғйҷҗпјҢжңҚеҠЎеҷЁйҖҡиҝҮ Web Push Protocol еҸ‘йҖҒеҠ еҜҶж¶ҲжҒҜпјҲйңҖ VAPID еҜҶй’ҘпјүпјҢWebз«ҜService Worker зҡ„ push дәӢд»¶и§ҰеҸ‘йҖҡзҹҘгҖӮ
  • дјҳзӮ№пјҡзҰ»зәҝжҺЁйҖҒгҖҒзұ»дјјеҺҹз”ҹеә”з”Ёзҡ„дҪ“йӘҢгҖӮ
  • зјәзӮ№пјҡйңҖ HTTPSгҖҒз”ЁжҲ·йңҖжҺҲжқғйҖҡзҹҘжқғйҷҗгҖҒе®һзҺ°иҫғеӨҚжқӮгҖӮ
  • йҖӮз”ЁеңәжҷҜпјҡжёҗиҝӣејҸ Web еә”з”ЁпјҲPWAпјүзҡ„зҰ»зәҝйҖҡзҹҘпјҲеҰӮйӮ®д»¶жҸҗйҶ’пјүгҖӮ

йҖҡиҝҮд»ҘдёҠеҗ„ж–№жЎҲзҡ„еҜ№жҜ”пјҢжҲ‘д»¬еҸҜд»ҘзҹҘйҒ“еҜ№дәҺз®ҖеҚ•дҪҺйў‘зҡ„жӣҙж–°пјҢдҪҝз”ЁиҪ®иҜўжҲ–й•ҝиҪ®иҜўеҚіеҸҜгҖӮе®һж—¶жҖ§иҰҒжұӮй«ҳдјҳе…ҲйҖү WebSocketпјҲеҸҢеҗ‘пјүжҲ– SSEпјҲеҚ•еҗ‘пјүгҖӮжңүзҰ»зәҝжҺЁйҖҒйңҖжұӮеҸҜд»ҘдҪҝз”Ё Service Worker + Push APIгҖӮжҲ‘д»¬иҰҒе®һзҺ°зҡ„зҪ‘йЎөзүҲжң¬жӣҙж–°еј№зӘ—еұһдәҺжҜ”иҫғз®ҖеҚ•дҪҺйў‘зҡ„жӣҙж–°пјҢеҸҜиғҪдёҖе‘ЁжҲ–иҖ…жӣҙй•ҝж—¶й—ҙжүҚдјҡжӣҙж–°дёҖдёӘзүҲжң¬пјҢжүҖд»ҘиҝҷйҮҢдҪҝз”ЁиҪ®иҜўжҲ–й•ҝиҪ®иҜўе°ұеӨҹдәҶгҖӮ

й•ҝиҪ®иҜўйңҖиҰҒжңҚеҠЎз«Ҝж”ҜжҢҒпјҢеҲ«дәәжңӘеҝ…жңүз©әжҗӯзҗҶжҲ‘д»¬пјҢд№ҹдёҚиҖғиҷ‘гҖӮеү©жңҖеҗҺдёҖдёӘж–№жЎҲпјҢиҪ®иҜўпјҢйӮЈд№ҲеҰӮдҪ•еңЁдёҚйңҖиҰҒжңҚеҠЎз«Ҝж”ҜжҢҒзҡ„жғ…еҶөдёӢпјҢе®һзҺ°иҝҷдёӘеҠҹиғҪе‘ўпјҹзӯ”жЎҲжҳҜпјҢеүҚз«ҜиҮӘе·ұе®ҡд№үдёҖдёӘ json ж–Үд»¶пјҢжҜҸж¬ЎиҪ®иҜўйғҪеҺ»иҜ·жұӮиҝҷд»Ҫж–Үд»¶пјҢеңЁжү“еҢ…жһ„е»әйҳ¶ж®өеҺ»жӣҙж–°ж–Үд»¶йҮҢйқўзҡ„зүҲжң¬зӣёе…іеӯ—ж®өпјҢиҝҷж ·йғЁзҪІе®ҢжҲҗеҗҺпјҢжҲ‘д»¬иҜ·жұӮеҲ°зҡ„жңҖж–°зүҲжң¬еҸ·е°ұжҳҜжңҖж–°зҡ„дәҶгҖӮ

ж¶ҲжҒҜжҺЁйҖҒзҡ„ж–№жЎҲжҲ‘д»¬жңүдәҶпјҢжҺҘзқҖжҳҜеҰӮдҪ•еҺ»жӣҙж–°зҪ‘йЎөе‘ўпјҹжҲ‘д»¬зҹҘйҒ“жңүж—¶еҖҷеҲ·ж–°дёҖдёӢпјҢзҪ‘йЎөиҝҳжҳҜж—§зҡ„пјҢиҝҷе°ұж¶үеҸҠеҲ°жөҸи§ҲеҷЁзҡ„зј“еӯҳзӯ–з•ҘдәҶгҖӮ

дәҢгҖҒзҪ‘йЎөжӣҙж–°ж–№жі•жҰӮиҝ°

еңЁж—ҘеёёејҖеҸ‘дёӯжңүж—¶еҖҷдјҡйҒҮеҲ°дёҖдёӘй—®йўҳпјҢе°ұжҳҜжҲ‘зҡ„д»Јз ҒжҳҺжҳҺе·Із»ҸжҺЁеҲ°жңҚеҠЎеҷЁдёҠйғЁзҪІеҘҪдәҶпјҢеҲ·ж–°з•ҢйқўпјҢиҝҳжҳҜж—§зҡ„пјҢжңүж—¶еҶҚеҲ·ж–°дёҖдёӢе°ұжҳҜж–°зҡ„йЎөйқўдәҶпјҢдҪҶжҳҜжңүж—¶еҲ·ж–°еҫҲеӨҡж¬Ўд№ҹиҝҳжҳҜж—§зҡ„з•ҢйқўпјҢиҝҷйҮҢжҲ‘д»¬еҸҜиғҪдјҡжғіеҲ°еҺ»жҠ“еҢ…пјҢжҹҘзңӢжҳҜеҗҰиҝ”еӣһдәҶ 304, жҳҜзҡ„иҜқпјҢжҲ‘д»¬еҶҚиҜ•иҜ• ctrl+f5гҖӮ

иҝҷйҮҢз•ҢйқўжІЎжӣҙж–°пјҢжҲ‘д»¬еҒҡзҡ„дёӨд»¶дәӢжғ…зҡ„дҫқжҚ®е°ұжҳҜжөҸи§ҲеҷЁзҡ„зј“еӯҳжңәеҲ¶гҖӮз¬¬дёҖж¬ЎеҲ·ж–°з•ҢйқўпјҢеҸҜиғҪе‘ҪдёӯдәҶжөҸи§ҲеҷЁзҡ„ејәзј“еӯҳпјҢиө„жәҗж–Үд»¶иҝҳжҳҜжӢҝзҡ„жң¬ең°зҡ„зј“еӯҳпјҢж №жң¬жІЎжңүеҺ»иҜ·жұӮжөҸи§ҲеҷЁиө„жәҗгҖӮжүҖд»ҘжҲ‘д»¬ f5 еҲ·ж–°дёҖдёӢпјҢдјҒеӣҫи·іиҝҮејәзј“еӯҳгҖӮи·іиҝҮејәзј“еӯҳжңүж—¶еҖҷз•ҢйқўиҝҳжҳҜж—§зҡ„пјҢеҸҜиғҪе‘ҪдёӯдәҶеҚҸе•Ҷзј“еӯҳпјҢиҝ”еӣһ 304пјҢйӮЈжҲ‘д»¬е°ұиҜ•иҜ•з”Ё ctrl+f5 и·іиҝҮд»–пјҢзӣҙжҺҘжӢҝжңҚеҠЎеҷЁдёҠзҡ„иө„жәҗгҖӮ
