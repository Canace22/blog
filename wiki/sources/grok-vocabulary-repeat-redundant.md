## 摘要

会话中给出两条英汉词义对照：**「重复」** 与 **Repeat**，**redundant** 与 **「多余的」**，并附简短释义与例句，便于在指令、界面文案等场景选用更贴切的词。

## 摘录要点

### 1. 重复 → Repeat

- **释义**：汉语「重复」作动词或名词，表示「再做一次」或「重复这一现象」；译为 **Repeat** 可保持简短、祈使语气（软件、音乐、日常指令等）。
- **例句（摘录）**：请不要重复同样的错误。 / Please do not repeat the same mistake.

### 2. redundant → 多余的

- **释义**：英语 **redundant** 表示不必要、超出所需、或信息重复；一般语境下译为 **「多余的」**；技术语境里亦可作 **「冗余的」**（工程、数据等）。
- **例句（摘录）**：This paragraph contains a lot of redundant information. / 这段文字包含很多多余的信息。
