推荐阅读主轴（一条线串起来）

  1. 从浏览器地址栏输入 URL 到界面被渲染出来看前端知识
  2. HTTP 笔记 · 说说浏览器缓存机制 · 认识 Cookie
  3. JS 概谈 · JavaScript 在 V8 引擎浏览器上是怎么执行的 · javaScript-promise
  4. CSS 可从 CSS 系列文章——CSS 基础知多少 起，串 CSS 常用布局及解决方案
  5. 框架：Vue 见 Vue 数据驱动视图更新实现、Vue3 实战笔记；Web 前端开发为什么要用框架
  6. 网络与安全：关于跨域那些事、Web安全 —— XSS攻击；CSRF 见 web 安全——CSRF
  7. 交付：webpack、前端性能优化、首屏时间(FCP) VS 白屏时间(FP)
