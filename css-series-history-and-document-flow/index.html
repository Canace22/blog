一、CSS 简史

1、1994 年被提议

2、1996 年 W3C 正式推出 CSS1

3、1998 年 W3C 正式推出 CSS2，推行内容和表现分离，table 布局开始没落

4、2007 年发布 CSS2.1，IE8 开始全面支持 CSS2.1

5、CSS3 在 CSS 2.1 的基础上进行改进，布局方法更多样(flex,grid,媒体查询等），视觉表现更丰富

二、CSS1 及 CSS2 的目的

为图文信息展示服务

三、文档流(流)

1、是什么

元素的基本定位和布局机制

2、文档流是如何影响元素布局的

(1) 正常文档流

(2) 破坏文档流，实现特殊布局

(3) 改变文档流，实现更加丰富的布局效果

3、流体布局

“div+css” 布局，比如有名的双飞燕布局、圣杯布局等
