使用 Docker 配置前端环境

2022-02-16 | web | 阅读 | 227 字 | 1 分钟

一、大体思路

1、设置 shell 别名运行 docker 命令

  • alias — 设置 shell 别名
  • node — 在终端输入的名称
  • docker —  docker 命令
  • run — 执行容器镜像
  • -rm — 关闭时自动删除容器
  • it —创建交互式容器
  • -v “$PWD”:/app — 把当前目录挂在到容器的 app 目录下
  • -w /app — 设置容器的工作目录为/app
  • node:16-alpine — 所使用的容器镜像

2、运行 source ~/.zshrc

二、具体配置

1、设置 node 环境，下面的例子是设置 node@16的，可以自己根据版本号命名另外设置，设置完之后记得执行一下 source ~/.zshrc。
