SSH 端口号修改

2019-11-21 | 工程化与运维 | 构建与部署 | 阅读 | 57 字 | 1 分钟

1、登录搬瓦工

2、打开我的服务器，打开 root shell interactive

3、输入 vim /etc/ssh/sshd_config

4、点击 i 键，i 修改 Port 值

5、esc => :wq 保存

6、 reboot 重启服务器

至此，ssh 端口恢复
