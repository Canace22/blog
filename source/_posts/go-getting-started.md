1、[下载](https://golang.org/dl/)

2、测试是否安装成功, 并运行程序

(1) 新建 hello.go

```go
package main

import "fmt"

func main() {
	fmt.Printf("hello, world\n")
}
```

(2) 执行代码

```go
go run hello.go 
```

(3) 编译二进制文件

编译 hello.go, 生成一份 hello 文件，执行该文件, 会输出一行 "hello, world"

```go
go build hello.go
./hello
```

(2)(3) 任一步骤执行成功，说明安装成功了
